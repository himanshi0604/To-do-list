-To-Do List Web App
This project is a simple To-Do List application built using HTML, CSS, and JavaScript.
It allows users to add tasks, mark them as completed, delete tasks, and manage their task list efficiently.

The interface is designed with a modern UI style and colorful layout to make task management visually appealing.

-Features
Add new tasks
Press Enter to add a task quickly
Mark tasks as completed
High priority tasks appear at the top
Live task counter
Delete all tasks
Clear completed tasks
Complete all tasks with one button

-Task Counter
The app automatically tracks:
Total tasks
Pending tasks
Completed tasks
These values update dynamically whenever a task is added, completed, or removed.

-Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and UI design
JavaScript (Vanilla JS) – Task logic and functionality
Google Fonts (Nunito) – Typography

-Priority System
Users can assign priority to tasks:
High Priority ⚠ – Appears at the top of the list
Medium Priority ● – Default priority
Low Priority ●
